## How to clone this ?
1. First clone this repo
   ```shell
   $ git clone https://github.com/vanaklay/TGP_db_to_front.git
   ```
2. Run
   ```shell
   $ bundle install
   ```
3. Run
   ```shell
   $ shotgun -p 3000
   ```

If something doesn't work ... please contact us or try this :

```shell
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ shotgun -p 3000
```
